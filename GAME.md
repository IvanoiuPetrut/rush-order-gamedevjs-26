# Rush Order

A small 2D factory game. Drag items off a conveyor belt, feed them into assembly stations, then package and deliver them to cars for points.

## Loop

Belt → Assembly Station → Packager → Car (+4 pts)

Dropping or burning an item costs 1 point. Game over at 0.

## Stack

- **Kaplay.js** — game engine
- **TypeScript** + **Vite** — language & bundler

## Run

```bash
npm install
npm run dev
```
